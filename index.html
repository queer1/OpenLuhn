Description

The source code served as an example of how Luhn algorithm works, it is, by no means, a commercial grade product. It might contain errors or flaws, and it was created for demonstration purpose only.

More information can be find on Wikipedia:

http://en.wikipedia.org/wiki/Luhn_algorithm

Usage

Either enter a full 16-digit credit card number to verify or replace part of the digits with "#" characters for brute-forcing.
